# SFU UNICEF Website

Official website for the SFU UNICEF campus club. This site provides information about our mission, events, and ways to get involved, and serves as a central hub for students interested in supporting UNICEF initiatives at Simon Fraser University.

## Live Site
Deployed with Cloudflare: [unicefcampusclub.ca](https://unicefsfucampusclub.ca/)

## Tech Stack
- React  
- Tailwind CSS  
- Cloudflare (Deployment & Hosting)
